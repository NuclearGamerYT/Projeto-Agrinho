## 📌 Sobre o projeto

Este projeto é um **site interativo desenvolvido para o concurso Agrinho**, com o objetivo de representar de forma visual e educativa a conexão entre o campo e a cidade.

A aplicação apresenta conteúdos relacionados ao meio rural, produção agrícola e tecnologia, utilizando uma interface simples, moderna e acessível.

De forma geral, trata-se de um projeto educacional que busca transformar informação em uma experiência digital envolvente — característica essencial de projetos que têm como objetivo criar algo único e com propósito definido .

---

## 🎯 Objetivo

* Demonstrar a relação entre **campo e cidade**
* Promover **consciência sobre produção e sustentabilidade**
* Aplicar conhecimentos de **desenvolvimento web**
* Criar uma experiência **interativa e educativa**
